# Find All Credit Approvals

Returns all credit approvals.

**URL** : `http://localhost:8083/api/creditApprovals`

**Method** : `GET`

**Request Params**

## Success Response

**Code** : `200 OK`

**Sample Response Body** :

```json
[
  {
    "approval": true,
    "customerTcNumber": 0,
    "givenCreditAmount": 0,
    "id": 0
  },
  {
    "approval": true,
    "customerTcNumber": 0,
    "givenCreditAmount": 0,
    "id": 1
  }
]
```
