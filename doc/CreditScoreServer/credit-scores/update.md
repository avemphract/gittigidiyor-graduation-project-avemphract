# Update Credit Score

Update to given credit score.

**URL** : `http://localhost:8082/api/creditScores`

**Method** : `PUT`

**Sample Request Body**

```json
{
  "tcNumber": 2,
  "creditScore": 100
}
```

## Success Response

**Code** : `200 OK`

**Sample Response Body** :

```json

{
  "tcNumber": 2,
  "creditScore": 100
}
```
