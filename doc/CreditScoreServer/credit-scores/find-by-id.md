# Find Credit Scores By Id

Returns a person by id

**URL** : `http://localhost:8082/api/creditScores/:id`

**URL Parameters** : `id=[long]`

**Method** : `GET`

**Request Params**

## Success Response

**Code** : `200 OK`

**Sample Response Body** :

```json
{
  "tcNumber": 2,
  "creditScore": 100
}
```
