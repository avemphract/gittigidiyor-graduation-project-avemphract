# Find Customer Tc Id

Returns a customer by tc

**URL** : `http://localhost:8082/api/customer/tc/:tc`

**URL Parameters** : `tc=[long]`

**Method** : `GET`

**Request Params**

## Success Response

**Code** : `200 OK`

**Sample Response Body** :

```json
{
  "creditApprovalsId": [
    0
  ],
  "name": "string",
  "phoneNumber": 0,
  "salary": 0,
  "surname": "string",
  "tcNumber": 0
}
```
