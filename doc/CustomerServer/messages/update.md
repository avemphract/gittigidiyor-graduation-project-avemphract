# Update Message

Update to message.

**URL** : `http://localhost:8083/api/messages`

**Method** : `PUT`

**Request Params**

```json
{
  "context": "string",
  "id": 0,
  "toPhone": 0,
  "type": "CREDIT_APPROVAL_RESULT"
}
```

## Success Response

**Code** : `200 OK`

**Sample Response Body** :

```json
{
  "context": "string",
  "id": 0,
  "toPhone": 0,
  "type": "CREDIT_APPROVAL_RESULT"
}
```
